Frequently asked questions

Is this a diagnostic tool?

No. CancerMamo is an assistive tool and should not be used as the sole basis for diagnosis. A qualified clinician must review results.

Can I upload DICOM?

Yes with server integration. This front-end demo accepts images; connect your backend to parse DICOM and return overlays.

How do you protect patient data?

We employ encrypted transport, short-lived URLs, and configurable retention. Remove PHI in demo mode.

Which model do you use?

The UI is model-agnostic (e.g., ResNet/ViT). Your backend exposes an endpoint that returns scores and a heatmap image.

Can I export results?

Yes. Add a server endpoint to generate PDF summaries with your logo and the reviewer’s notes.

Is there a dark mode?

You’re looking at it. The palette is optimized for low-glare reading.
